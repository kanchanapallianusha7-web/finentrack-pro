# Personal Expense Management System with Data Visualization Dashboard

## 📌 Description
A full-stack web application designed to help users efficiently track and manage their expenses. The system includes secure user authentication, expense categorization, and an interactive data visualization dashboard for analyzing spending patterns.

## 🚀 Features
- 🔐 Login / Signup system (secure authentication)
- 📊 Dashboard with total income, expenses, and balance
- ➕ Add, edit, and delete expenses (CRUD operations)
- 🗂️ Category-based expense tracking (Food, Travel, Shopping, etc.)
- 📈 Bar graph for comparative analysis
- 🥧 Pie chart for category-wise distribution
- 📉 Monthly expense line graph for trend analysis
- ⚠️ Budget limit alerts
- 🔍 Filter data (daily, monthly, yearly)
- 📋 Detailed expense table
- 📤 Export reports to PDF and Excel

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (LocalStorage)
- Chart.js
- jsPDF & jsPDF-AutoTable
- SheetJS (xlsx)

## ▶️ How to Run
1. Clone or download this repository  
2. Open `login.html` in your browser  
3. Register a new account and log in  
4. Start adding expenses and view insights on the dashboard
